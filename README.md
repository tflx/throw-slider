# throw-slider
Slider with throw effect

All credit goes to GSAP https://codepen.io/GreenSock/pen/ExdyYed

---


## Usage

### Install

```
npm i throw-slider
```

### Import

```
import 'throw-slider'
```

### Use

```

    <div>
        <img src="https://picsum.photos/800/600?random=1" alt="slide 1">
```
